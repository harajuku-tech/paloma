Running Script
=======================

Setup
-------

::

    sudo cp paloma.sh /etc/init.d/paloma

Configure
------------

::

    sudo vim /etc/init.d/paloma


Install
-----------

::

    sudo /etc/init.d/paloma install

Uninstall
-----------

::

    sudo /etc/init.d/paloma uninstall
    sudo rm /etc/init.d/paloma
